# luit-feb-2026-python

Python practice repo for the Level Up In Tech (LUIT) February 2026
cohort.

## What's in this repo

-   `hello_world.py` --- a simple "Hello, World!" script.
-   An imports demo script that uses:
    -   `random`
    -   `math`
    -   `datetime`
    -   `statistics`
    -   `matplotlib` (creates a histogram)

## How to run

From the repo root:

``` bash
python hello_world.py
```

Run the imports demo script (replace with the actual filename you used):

``` bash
python using_imports.py
```

## Notes

-   The imports demo script opens a matplotlib window to display a chart
    when it runs.
